# 🚚 Supply Chain & Delivery Performance Dashboard (Power BI)

## 📌 Project Overview

This project analyzes supply chain performance and delivery efficiency using Power BI. It provides insights into supplier performance, delivery delays, and logistics optimization.

---

## 🎯 Objective

* Monitor delivery performance
* Identify delayed shipments
* Evaluate supplier efficiency

---

## 📊 Key Features

* On-time vs delayed deliveries
* Supplier performance analysis
* Delivery time trends
* KPI metrics for logistics

---

## 🛠️ Tools Used

* Power BI
* DAX
* Data Modeling

---

## 🔍 Key Insights

* Certain suppliers consistently delay deliveries
* Peak delays occur during specific time periods
* On-time delivery rate varies across regions
